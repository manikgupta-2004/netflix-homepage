# 📖 About The Project

This project is a responsive clone of the Netflix homepage built using **HTML5** and **CSS3**.

The objective of this project was to improve frontend development skills by recreating one of the world's most recognizable streaming platforms while following clean coding practices and responsive web design principles.

---

# ✨ Features

- 🎬 Netflix-inspired UI
- 📱 Fully Responsive Design
- 🎨 Modern Layout
- ⚡ Clean HTML Structure
- 🎯 CSS Flexbox
- 📦 CSS Grid
- 🖼 Hero Banner
- 📺 Multiple Content Sections
- 📥 Footer Section
- 💻 Cross Browser Compatible
